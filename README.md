# 💰 Manager Wydatków SQL

Mój pierwszy projekt w Pythonie, który pomaga ogarnąć domowy budżet przy użyciu bazy danych SQLite.

## 🚀 Co potrafi ten program?
* Dodaje wydatki do bazy danych.
* Sumuje wszystkie wpisy.
* Pozwala zachować historię wydatków nawet po zamknięciu programu.

## 🛠️ Jak to odpalić?
W terminalu wpisz:
```bash
python3 Wydatek.py
